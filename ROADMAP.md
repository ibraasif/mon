# Roadmap

## v0 (current)
- `POST /ingest` — store + embed a thought
- `GET /search?q=` — semantic recall
- MCP tools: `recall_memory`, `save_memory`
- MCP transport: Streamable HTTP (JSON-RPC 2.0)
- Bearer token auth on all endpoints
- claude.ai connector support
- Claude Desktop support via `mcp-remote`

## v1 ideas
- Telegram bot capture integration
- Tags and metadata on thoughts
- Multi-user support
- Durable Objects for persistent SSE (enables Perplexity support)

## v2 ideas
- Content pipeline integration
- Dashboard UI
- Hosted SaaS layer
